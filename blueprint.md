
# pers. 프로젝트 청사진

## 1. 프로젝트 개요

`pers.`는 사용자가 자신의 개성을 발견하고 표현할 수 있도록 돕는 종합 뷰티/패션 플랫폼입니다. 향수, 네일, 패션을 아우르며, 각 분야의 깊이 있는 정보와 개인화된 추천, 그리고 커뮤니티 기능을 제공합니다.

## 2. 구현된 기능 및 디자인

### 버전 1.0: 향수 섹션 기반 구축

*   **디자인:**
    *   전체적으로 미니멀하고 깨끗한 디자인을 채택합니다.
    *   폰트는 'Pretendard'를 사용하여 가독성과 미려함을 높입니다.
    *   메인 컬러는 화이트와 라이트 그레이를 사용하고, 포인트 컬러로 블랙을 사용합니다.
    *   'for your **pers**onality' 슬로건에서 'pers'는 검정색, 나머지는 회색으로 강조합니다.
*   **레이아웃:**
    *   헤더: 로고, 네비게이션 (향수란?, 향수 알아보기, 커뮤니티), 아이콘 (검색, 북마크, 프로필)
    *   Hero 섹션: 메인 이미지와 슬로건
    *   콘텐츠 섹션: 3단 그리드로 각 카테고리 (향수란?, 향수 알아보기, 커뮤니티) 소개

## 3. 현재 작업 계획

### 버전 1.1: '향수 가이드' 페이지 구축

1.  **'향수 가이드' 페이지 (`perfume-guide.html`) 생성:**
    *   '향수란?' 메뉴 클릭 시 이동할 상세 정보 페이지를 신규 생성합니다.
    *   페이지를 **2단 레이아웃**으로 구성합니다.
        *   **왼쪽:** 고정된 하위 네비게이션 메뉴 (향수의 정의, 향의 계열, 부향률에 따른 분류, 올바른 사용 가이드, 사용 시 주의사항)
        *   **오른쪽:** 각 메뉴에 해당하는 상세 콘텐츠 섹션
    *   '향수의 정의' 섹션에 사용자가 제공한 내용을 기반으로 세련되게 다듬은 텍스트를 추가합니다.

2.  **스타일링 업데이트 (`style.css`):**
    *   `perfume-guide.html`의 2단 레이아웃, 하위 네비게이션, 콘텐츠 섹션에 대한 스타일을 추가합니다.
    *   하위 네비게이션의 활성화된(현재 보이는) 메뉴 항목에 적용할 스타일(굵은 글씨)을 정의합니다.

3.  **인터랙션 구현 (`main.js`):**
    *   **스크롤 스파이(Scroll Spy):** 사용자가 페이지를 스크롤할 때, 화면에 보이는 콘텐츠 섹션에 맞춰 왼쪽 하위 메뉴의 활성화 상태가 자동으로 변경되는 기능을 구현합니다.
    *   **부드러운 스크롤(Smooth Scroll):** 하위 메뉴 클릭 시 해당 콘텐츠 섹션으로 부드럽게 이동하는 기능을 추가합니다.

4.  **메인 페이지 링크 수정 (`index.html`):**
    *   '향수란?' 카드의 링크를 새로 만든 `perfume-guide.html` 페이지로 연결합니다.
