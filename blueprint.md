
# pers. 프로젝트 청사진

## 1. 프로젝트 개요

`pers.`는 사용자가 자신의 개성을 발견하고 표현할 수 있도록 돕는 종합 뷰티/패션 플랫폼입니다. 향수, 네일, 패션을 아우르며, 각 분야의 깊이 있는 정보와 개인화된 추천, 그리고 커뮤니티 기능을 제공합니다.

## 2. 구현된 기능 및 디자인

### 버전 1.0: 향수 섹션 기반 구축

*   **디자인:**
    *   전체적으로 미니멀하고 깨끗한 디자인을 채택합니다.
    *   폰트는 'Pretendard'를 사용하여 가독성과 미려함을 높입니다.
    *   메인 컬러는 화이트와 라이트 그레이를 사용하고, 포인트 컬러로 블랙을 사용합니다.
    *   'for your **pers**onality' 슬로건에서 'pers'는 검정색, 나머지는 회색으로 강조합니다.
*   **레이아웃:**
    *   헤더: 로고, 네비게이션 (향수란?, 향수 알아보기, 커뮤니티), 아이콘 (검색, 북마크, 프로필)
    *   Hero 섹션: 메인 이미지와 슬로건
    *   콘텐츠 섹션: 3단 그리드로 각 카테고리 (향수란?, 향수 알아보기, 커뮤니티) 소개
    *   상세 정보 섹션: 향의 종류 등 이미지 기반의 그리드 목록

## 3. 현재 작업 계획

1.  **기본 HTML 구조 설정 (`index.html`):**
    *   시맨틱 태그 (`header`, `main`, `nav`, `section`)를 사용하여 페이지의 기본 구조를 잡습니다.
    *   Google Fonts에서 'Pretendard' 폰트를 가져옵니다.
    *   헤더, Hero 섹션, 3개의 기본 콘텐츠 섹션을 생성합니다.

2.  **기본 스타일링 (`style.css`):**
    *   전체적인 폰트, 색상, 여백을 설정합니다.
    *   Flexbox와 Grid를 사용하여 헤더와 콘텐츠 섹션의 레이아웃을 구성합니다.
    *   Hero 섹션의 슬로건(`for your personality`) 스타일을 디자인에 맞게 구현합니다.

3.  **JavaScript 파일 생성 (`main.js`):**
    *   미래의 인터랙티브 기능을 위해 빈 JavaScript 파일을 생성하고 HTML에 연결합니다.
